# VoltSynth

VoltSynth is a browser-based synthesizer built with TypeScript and the Web Audio API.

It is Phase 1 of Operation VoltAudio, a larger project focused on building web-based audio tools including a synth, sequencer, and lightweight DAW.

## Planned Features

- Keyboard note mapping
- Polyphonic note playback
- ADSR envelope controls
- Waveform selection
- Low-pass filter controls
- Clean interactive UI
- Browser-based audio engine

## Tech Stack

- TypeScript
- Vite
- Web Audio API
- HTML/CSS

## Development Status

VoltSynth now has a first working app screen. The project has a working TypeScript/Vite foundation, GitHub repository, GitHub Pages deployment, Hackatime tracking, and a minimal Web Audio synth interface.

Current implementation:
- Start Audio control
- Play Test Note control
- Basic Web Audio oscillator engine
- Waveform selector
- Output level slider
- Clickable one-octave note strip
- Clean dark interface

Next steps:
